# Roadmap — Globe Interactif 3D

Date : 18 juin 2026

## Vision générale

Créer progressivement une application interactive d’exploration du monde.

La priorité est de construire une V1 simple, stable et présentable, puis d’ajouter des fonctionnalités plus ambitieuses.

## Version 0 — Préparation

Objectif :
préparer le projet et les documents.

Tâches :

- créer les documents de conception ;
- définir le périmètre ;
- définir les technologies ;
- préparer le dépôt GitHub ;
- créer le fichier AGENTS.md.

Statut :
à faire avant développement.

## Version 1 — Globe interactif fonctionnel

Objectif :
avoir une première version utilisable.

Fonctionnalités :

- projet React Vite ;
- globe 3D ;
- rotation ;
- zoom ;
- affichage des pays ;
- frontières ;
- survol pays ;
- clic pays ;
- fiche pays ;
- recherche pays ;
- labels océans principaux ;
- design responsive ;
- build fonctionnel ;
- déploiement GitHub Pages.

Résultat attendu :
une démo propre et présentable.

## Version 1.1 — Amélioration UX

Objectif :
rendre l’expérience plus agréable.

Fonctionnalités :

- animations plus fluides ;
- amélioration mobile ;
- meilleure infobulle pays ;
- meilleurs messages d’erreur ;
- optimisation du chargement ;
- ajout d’un écran d’aide.

## Version 1.2 — Données enrichies

Objectif :
ajouter plus d’informations.

Fonctionnalités :

- drapeaux ;
- population ;
- superficie ;
- langue ;
- monnaie ;
- capitale ;
- description plus complète ;
- liens internes ;
- alias de pays.

## Version 2 — Mers et océans interactifs

Objectif :
rendre les mers et océans plus présents.

Fonctionnalités :

- labels plus précis ;
- fiche océan ;
- fiche mer ;
- clic sur les labels ;
- informations sur superficie, profondeur moyenne, pays riverains si disponible.

## Version 2.1 — Mode éducatif

Objectif :
transformer le projet en outil d’apprentissage.

Fonctionnalités :

- quiz pays ;
- quiz capitales ;
- quiz continents ;
- score ;
- mode entraînement ;
- mode révision.

## Version 2.2 — Marques et points d’intérêt

Objectif :
ajouter des marqueurs sur le globe.

Fonctionnalités :

- capitales affichées sur le globe ;
- grandes villes ;
- monuments ;
- volcans ;
- montagnes ;
- zones climatiques.

## Version 3 — Backend optionnel

Objectif :
ajouter un backend si le projet devient plus grand.

Technologie possible :

- Symfony ;
- API Platform ;
- base de données ;
- espace admin.

Fonctionnalités possibles :

- gestion des fiches pays ;
- ajout de contenus depuis un back-office ;
- favoris utilisateur ;
- statistiques ;
- sauvegarde de progression quiz.

## Version 4 — Expérience avancée

Objectif :
rapprocher l’expérience d’une application cartographique complète.

Fonctionnalités possibles :

- mode satellite amélioré ;
- couches thématiques ;
- distances entre points ;
- fuseaux horaires ;
- comparateur de pays ;
- mode plein écran ;
- export de fiches ;
- partage d’un pays par lien.

## Priorité actuelle

La priorité absolue reste :

1. globe visible ;
2. interactions de base ;
3. pays cliquables ;
4. fiche pays ;
5. recherche ;
6. responsive ;
7. GitHub Pages.

Ne pas partir trop vite sur les fonctionnalités avancées avant que la V1 soit stable.
